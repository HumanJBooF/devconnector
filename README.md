# devconnector
Social app for developers built with MERN stack

### Features
* Profiles
* Posts
* Likes/Dislikes
* Password protected with bcrypt
* Gravatar avatar

### Will be adding
* Pc/Tech news section with sharing and comments
* Option to upload your own avatar
* Friends list
* Chat

### Built With
* React
* Redux
* Bootstrap4
* Node.js
* Express.js
* Passport.js
* Bcrypt
* JSON Web Token
* MongoDB
* Mongoose
* Axios
